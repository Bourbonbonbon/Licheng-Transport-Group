## 🆘 常见问题

### Q: 如何修改管理密码？
A: 在代码中找到 `DEFAULT_PASSWORD` 变量，修改为你的新密码。

### Q: 数据丢失了怎么办？
A: 所有数据存储在浏览器本地，清除浏览器数据会丢失。建议定期备份。

### Q: 支持哪些图片格式？
A: 支持 JPG、JPEG、PNG 格式。

### Q: 可以部署到服务器吗？
A: 可以，直接上传HTML文件到任何静态托管服务即可。
